# 电梯电费公摊计算系统 — 项目说明书

## 1. 项目概述

为出租工业园（A栋、B栋）开发一个网页应用，用于计算各租户每月的电梯电费公摊。

- **楼栋**：A栋、B栋，每栋 6 层，2-6 楼租户使用电梯
- **电梯**：A1、A2、A3（A栋），B1、B2、B3（B栋）
- **核心需求**：管理员管理基础资料；普通用户每月录入电表读数后自动计算各租户电梯公摊电费

---

## 2. 用户角色与权限

| 角色 | 权限 |
|---|---|
| 管理员 (admin) | 全部：管理租户资料、录入电表、查看账单、导出/打印、修改电费单价 |
| 普通用户 (user) | 仅：录入电表读数、查看账单、导出/打印（不可修改基础资料） |

---

## 3. 数据模型

### 3.1 用户表 `users`
| 字段 | 类型 | 说明 |
|---|---|---|
| id | INTEGER | 主键 |
| username | TEXT | 登录名 |
| password_hash | TEXT | bcrypt 哈希 |
| role | TEXT | `admin` 或 `user` |

### 3.2 租户表 `tenants`
| 字段 | 类型 | 说明 |
|---|---|---|
| id | INTEGER | 主键 |
| name | TEXT | 租户名称 |
| building | TEXT | `A` 或 `B` |
| floor | INTEGER | 楼层 2-6 |
| area | REAL | 租赁面积（㎡） |
| elevators | TEXT | 使用的电梯，逗号分隔，如 `A1,A2` |

### 3.3 系统配置表 `settings`
| 字段 | 类型 | 说明 |
|---|---|---|
| key | TEXT | 配置键，如 `unit_price` |
| value | TEXT | 配置值 |

### 3.4 电表读数表 `meter_readings`
| 字段 | 类型 | 说明 |
|---|---|---|
| id | INTEGER | 主键 |
| elevator | TEXT | 电梯编号：A1/A2/A3/B1/B2/B3 |
| year_month | TEXT | 年月，格式 `YYYY-MM` |
| reading | REAL | 电表读数（kWh） |

### 3.5 账单表 `bills`
| 字段 | 类型 | 说明 |
|---|---|---|
| id | INTEGER | 主键 |
| tenant_id | INTEGER | 关联租户 |
| year_month | TEXT | 年月 |
| total_cost | REAL | 总公摊电费（元） |
| breakdown | TEXT | JSON 明细：每部电梯分摊金额 |

## 4. 计算逻辑

以某月为例（本月 = 2026-06，上月 = 2026-05）：

```
1. 电梯本月用电量 = 本月读数 - 上月读数
2. 电梯本月电费 = 用电量 × 全局电费单价
3. 电梯每㎡单价 = 电梯本月电费 ÷ 使用该电梯的所有租户面积之和
4. 租户公摊电费 = 租户面积 × (所用电梯1的㎡单价 + 所用电梯2的㎡单价 + …)
```

**示例**：A栋某租户 500㎡，使用 A1、A2。A1 单价 0.30 元/㎡，A2 单价 0.25 元/㎡ → 该租户当月电费 = 500 × 0.55 = 275 元。

## 5. 页面结构

| 页面 | 访问角色 | 功能 |
|---|---|---|
| 登录页 | 全部 | 用户名+密码登录，JWT 鉴权 |
| 租户管理 | 管理员 | 增删改查租户：名称、楼栋、楼层、面积、使用电梯 |
| 电表录入 | 全部 | 选年月，填 6 部电梯本月读数，自动计算 |
| 账单查看 | 全部 | 选年月，查看各租户电费明细，导出 Excel / 打印 |
| 系统设置 | 管理员 | 修改全局电费单价 |

## 6. 技术栈

| 层 | 技术 |
|---|---|
| 前端 | React + Vite + React Router |
| 后端 | Cloudflare Workers（Hono 框架） |
| 数据库 | Cloudflare D1（SQLite） |
| 部署 | Cloudflare Pages（前端）+ Workers（API） |
| 导出 | SheetJS（xlsx）前端生成 Excel |
| 鉴权 | JWT + bcrypt |

---

## 7. 初始化

系统首次部署时，D1 数据库预置：
- 一个 admin 账号（用户名/密码在部署时配置）
- 全局电费单价默认值（管理员可修改）
